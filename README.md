## üíª Getting Started

To run this portfolio locally:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/ajay-automates/ajaykumarreddynelavetla-portfolio.git
    cd ajaykumarreddynelavetla-portfolio
    ```

2.  **Install dependencies:**
    ```bash
    npm install --legacy-peer-deps
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```
