## Why This Exists

Version 2 of my portfolio — rebuilt from scratch with Vite for instant HMR, TypeScript for type safety, and a component architecture designed for easy content updates. Optimized for speed and SEO.

---

## Features

| Feature | Description |
|---------|-------------|
| **Instant Dev Server** | Vite's lightning-fast HMR — sub-50ms hot reload |
| **Type-Safe** | Full TypeScript throughout components and data |
| **Responsive** | Tailwind CSS mobile-first responsive design |
| **Accessible** | Semantic HTML with ARIA attributes |
| **Optimized Build** | Tree-shaking, code splitting, lazy loading |
| **GitHub Actions** | Automated deployment pipeline |

## Quick Start

```bash
git clone https://github.com/ajay-automates/ajaykumarreddynelavetla-portfolio.git
cd ajaykumarreddynelavetla-portfolio
npm install    # or: bun install
npm run dev    # or: bun dev
# Open http://localhost:5173
```

### Build for Production

```bash
npm run build
npm run preview
```

## Key Design Decisions

| Decision | Rationale |
|----------|-----------|
| **Vite over Next.js** | Pure SPA — no SSR needed, faster builds |
| **Bun support** | 3x faster installs, compatible with npm |
| **TypeScript strict** | Catch bugs at compile time |
| **Tailwind over CSS-in-JS** | Zero runtime cost, consistent design tokens |
